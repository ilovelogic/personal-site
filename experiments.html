Extra Credit: MathML

Einstein's mass-energy equivalence, written with native MathML instead of an image or a LaTeX-rendering script:

E = m c 2

MathML has been part of the HTML specification since HTML5, but developers could not rely on it in production for most of that time. Internet Explorer never supported it at all, and Chrome's story was worse than simply "missing": Chrome 24 shipped MathML support, then pulled it back out in Chrome 25 because the implementation was unstable, leaving it absent from Chrome for roughly seven years (through Chrome 96). Firefox was the one steady exception, supporting MathML since Firefox 2. With that much fragmentation across the dominant browser, teams had to fall back on image snapshots of formulas or JavaScript rendering libraries such as MathJax or KaTeX instead of the native tag. Chrome only regained full, stable MathML support in version 109 (released January 2023), and Edge followed the same timeline since it is Chromium-based. That means reliable cross-browser native MathML is really only about two years old as of this writing, which is why a tag that has existed since the earliest HTML5 drafts still feels new today.
